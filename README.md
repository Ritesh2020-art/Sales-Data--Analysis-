# Excel Sales Dashboard Project

## Overview

This project analyzes sales data using Microsoft Excel. A dashboard was created using Pivot Tables and charts to visualize sales performance across different regions and product categories.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning

## Dataset

The dataset contains 3000 sales records with the following columns:

* Order ID
* Date
* Month
* Region
* Product Category
* Units Sold
* Unit Price
* Total Amount

## Key Analysis

* Region wise sales performance
* Product category comparison
* Monthly sales trends
